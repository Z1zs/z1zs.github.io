MathAgent is a Mixture-of-Math-Agent framework for detecting errors in multimodal mathematical problems. It routes image/text reasoning workflows across specialized agents and integrates external tools for robust problem analysis.

Highlights:

- Designed three specialized agents for multimodal error detection and dynamic routing.
- Built MathToolKits with OCR, symbolic computation, and diagram parsing.
- Reduced API calls by 40% through automated tool debugging and latency-aware scheduling.
- Deployed the framework in production with 98% API success rate and 89.3% student satisfaction across 1M+ K-12 users.

This work was recognized as an ACL 2025 Industry Oral.
