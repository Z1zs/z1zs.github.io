MemoryOS is an open-source memory layer for AI assistants and LLM-based agents. It focuses on personalized interaction, long-term context management, and memory operations that can be integrated into practical agent systems.

Key contributions:

- Built support for long-term memory management across agent workflows.
- Integrated with popular LLM frameworks and protocols, including Ollama, OpenAI, vLLM, and MCP.
- Supported multimodal memory across text, images, videos, and audio.
- Benchmarked the system against competing memory-layer products such as Mem0 and Zep Cloud.
